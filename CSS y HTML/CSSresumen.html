Estilo en linea de codigo

Una de las formas de poder aplicar estilos a nuestra pagina es hacerlo directamente desde la linea de codigo. Este estilo debemos aplicarlo en la etiqueta de apertura a la cual lo aplicaremos en forma de atributo colocando 'style'. Dentro de las diferentes modificaciones que podemos hacer estan el color, la fuente, el tamano, la ubicacion en la pagina, etc.


Estilo en head

Otra de las maneras que tenemos de aplicar estilos a nuestra pagina es a travez de la etiqueta <'style'> colocada dentro de head. Dentro de esta etiqueta colocaremos la etiqueta, nombre, id, tipo o class que buscamos darle estilo y entre {} iremos modificando el color, la fuente, el tamano de la fuente, etc.

Por ej:

h1{
color: red;
font: arial;
font-size: 5px;
}



De esta manera, le aplicamos estilo al texto dentro de la etiqueta h1.
A diferencia del aplicado en linea, cuando utilizamos el style en head, este estilo se aplica a todos los textos que compartan etiqueta, por ej, si tenemos mas de un header 1, el estilo se aplicara a todos y cada uno de ellos.


Creando un archivo css

Otra de las maneras de aplicar estilo a una pagina web es creando un archivo .css en el cual, de la misma manera que aplicando stilos dentro de head, llamaremos a cada etiqueta, nombre, id, type, etc, de manera individual y mediante {} aplicaremos cambios de color, fuente, etc.
Para utilizar este archivo css, en la parte de head vamos a utilizar una nueva etiqueta llamada <'link'> seguida de un href en el cual colocaremos la direccion en nuestra pc donde se encuentre el archivo css a utilizar. Por ej: href="../CSS y HTML/style.css" rel="stylesheet"
Para evitar que todos los estilos se apliquen a la etiqueta a la cual llamamos, como por ejemplo h1, debemos aplicar estos estilos utilizando el nombre o el class de la etiqueta, el cual logra que estos cambios se apliquen especificamente a estos.


Selectores CSS

Los selectores son la forma en la cual nosotros seleccionamos el tipo de elemento a la cual se le va a aplicar el estilo indicado en nuestro archivo css.
Como ya indicamos ateriormente, contamos con varios selectores basicos.


Selector de tipo

Selecciona a los elementos por su tag name.
Por ej 'h1' selecciona a todos los elementos <'h1'>


Selector de clase

El selector de clase llama a los elementos por su class name y se indica de la siguiente manera: '.texto'. En este caso, llamaria a todos los elementos con el class name 'texto'.


Selector de ID

Selecciona al elemento basandose en su valor de atributo ID. Se indica de la siguiente manera: #nombre. Se aplicaria a todos los elementos con el valor de ID 'nombre'.


Selector universal

Selecciona a todos los elementos. Opcionalmente puede estar restringido a un espacio de nombre especifico o a todos los espacios de nombres.
Se aplica de la siguiente manera: *. Se aplicara a todos los elementos del documento.


Selector de atributo

Selecciona a los elementos basandose en el valor de un determinado atributo.
Ej: img[alt] Se aplicara a todas las imagenes que contengan el atributo 'alt'
Ej: [type="radio"] De esta manera seleccionamos a todos los elementos de tipo radio.


Selector de hijos directos

Entre las distintas formas que tenemos de llamar a un elemento para darle estilo tenemos a estos 3 selectores, los cuales primero toman la class del elemento principal y llaman a los hijos de este.
Para ser mas claro, vamos a aplicar un ej:

                             HTML
                            <'ul class="hijos-directos">
                                <'li>Ejemplo
                            <'/ul>

                            CSS
                        hijos-directos > li{
                            background-color: red;
                        }

De esta manera, el color de fondo se aplicaria al elemento li, el cual es hijo directo del elemento con la class hijos-directos.

Fuentes

Las fuentes son partes fundamentales de nuestro estilo de pagina, y la forma de aplicarla es mediante font-famyly. Tenemos varios tipos de familias de fuentes para seleccionar. Se pueden agregar diferentes tipos de fuentes por si la primer seleccion falla y estas se separan con una coma. Ej:
font-family: Verdana, 'Times New Roman', Gotic.


Google Fonts

Google nos ofrece una libreria con diferentes tipos de fuentes las cuales podemos descargar o utilizar de manera online mediante un <'link'>.
Primero debemos ingresar a Google Fonts y elegir la fuente deseada.
Una vez elegida la fuente, elegimos el estilo deseado y nos dara la opcion de copiar el link a agregar. Una vez copiado este link, lo agregamos al head de nuestro archivo html y luego copiamos el font family que nos ofrece google y lo agregamos a nuestro archivo css.


Personalizar bordes de imagenes

Las imagenes que colocamos dentro de nuestra web pueden ser personalizadas, como por ej cambiar el ancho[widht], el alto[height], podemos agregarle bordes, entre otras cosas.
El ultimo mencionado, el borde, tiene distintos tipos de personalizaciones. Primero comenzamos agregando el border-width, luego podemos agregarle un color, stilo, entre otras cosas. Un borde puede tener diferentes estilos en cada uno de sus lados y estos se agregan solamente poniendo el nombre del estilo separado por un espacio y se van a colocar en orden: arriba, derecha, abajo, izquierda.

Imagenes redondeadas

Dentro de todas las personalizaciones que podemos agregarle a nuestras imagenes, una de ellas es la de agregarle bordes redondeados. Si aprovechamos esta funcion, podemos lograr que nuestra imagen tome forma redondeada dandole un valor de 50% a este border-radius.

Padding

Padding es el espacio entre el contenido y el borde del elemento. El borde es escensial para conocer la distancia entre un elemento y otro. Podemos aplicar diferentes padding a cada lado del elemento con el atributo sumado a top, right, left or bottom. El tamano que apliquemos se aplicara al lado que elijamos.
Por ej: padding-top: 10px. (Se aplicaran 10pixeles a la parte superior).
Para poder aplicar padding a cada lado de nuestro elemento en una sola linea, podemos hacerlo todo dentro del elemento padding agregando los valores en sentido a las agujas del reloj, tomando top el primer valor, right el segundo, bottom el tercero y lefl el ultimo.
Por ej: padding: 10px 15px 5px 10px(se le aplicaran 10 pixeles a top, 15 pixeles a right, 5 pixeles a bottom y 10 pixeles a left).

Margin

Margin es la distancia entre el borde de un elemento con el borde de otro elemento.
Al igual que con padding, a margin se le pueden aplicar valores llamando a cada lado individualmente(top, right, left, bottom) o haciendolo en una sola linea(se aplica siguiendo las agujas del reloj).

Unidades Relativas y absolutas

Todas las propiedades que se utilizan en CSS tienen un valor o un conjunto de valores que esa propiedad admite.
Las unidades absolutas no son relativas a nada mas que a ellas mismas, si aplicamos el valor de unidad absoluta a un elemento, este tendra el valor designado y no va a variar.
Un ej de unidades absoluta es px(pixel).

Las unidades relativas dependen del elemento en el cual esten contenidas, como puede ser el tamano de la letra del elemento o a la ventana grafica.
De estas unidades vamos a prestar mayor atencion a vw y a em.
VW: vw es relativa al ancho de la ventana grafica, por lo que se ira modificando a la par que se modifique el ancho de la ventana del navegador.

EM: em es relativa al tamano de la letra que contiene el elemento padre, si por ej aumentamos el tamano de la letra, el elemento que contenga esta unidad tambien aumentara.

Colores hexadecimales

Un codigo hexadecimal es un codigo en base 16. En esta base tenemos numeros entre 0 y 9 y letras entre A y F entre 6 digitos.
EJ: #000000#. Entre estas posibilidades, se le aplica en los primeros dos lugares cuanto de rojo, en los segundos dos, cuanto de verde y en los terceros dos cuanto de azul tendra el color(#00 00 00#).
El mayor valor que se le puede agregar al color es F, ej : #FFFFFF#(blanco)
Si queremos buscar el codigo de algun color especifico, lo podemos hacer colocando 'color picker' en nuestro navegador.
En ciertos colores podemos abreviar nuestros codigos hexadecimales en 3 digitos. Cuando los digitos de cada par estan repetidos, se pueden abreviar de la siguiente manera: #00ff00# ---> #0f0#.

Variables en CSS

Para crear una variable, primero debemos colocar dos guiones(--) seguido del nombre de la variable y del valor de esta.
Una de las funcionalidades que podemos encontrarles es dandole un valor de color, por ej, --color: red. Luego cuando queramos asignar este color en alguna etiqueta, solamente colocamos el atributo, seguido de var y entre parentesis el nombre de la variable.
Ej: h2{
background-color: var(--color)
}
En este caso, el color de fondo de h2 seria el valor de la variable previamente asignado. Esto nos es muy util ya que, al asignarle el valor de nuestra variable a diferentes elementos, si buscamos cambiar el valor de la variable solamente modificaremos su valor y este se aplicara a todos los elementos.

Valores de respaldo

Para prevenir errores en la asignacion de variables, en caso de que la variable no se consiga o no este correcta, el valor de respaldo nos ayudaria a cubrir la ausencia del valor principal. Para agregarlo, solamente colocamos una coma, seguido de un espacio y el valor de respaldo.
Ej: h1{
background-color: var(--color, negro)
}

Pseudoclases

Una pseudoclase es una palabra clave que se anade a los selectores y que se especifica un estado especial del elemento seleccionado.
Por Ej: :valid aplicara un estilo a un formulario con una informacion valid, como por ej un formulario de email con un email correcto, en caso de que el email este incorrecto este estilo no se le aplicara.
Tenemos muchas pseudoclases, las cuales podemos ver dentro de este link. Para aplicar estilo utilizando la psuedoclase hay que hacerlo de la siguiente manera: 

                        .nombre-de-la-clase *:valid{
                            border-color: red;
                        }

Colores en CSS

Sistema hexadecimal: 0123456789ABCDEF
El sistema hexadecimal cuenta con 6 digitos, los cuales son combinaciones de las letras y los numeros, y representan cada dos digitos a los colores rojo, verde y azul respectivamente.
Ej: #FF6600

Colores RGB: Este tiene mucha relacion con el sistema hexadecimal, estos se aplican entre () separados con comas y espacios, y van desde el 0 al 255 por cada color.
Ej: rgb(255, 255, 255) -en este caso, seria color blanco-.

Sistema HSL(Hue Saturation Lightness - Tono Saturacio Luminocidad): Se considera como el sistema mas intuitivo.
El primer valor es HUE(Circulo cromatico):
0° -> red
60° -> yellow
120° -> green
180° -> cyan
240° -> blue
300° -> magenta

El segundo valor es Saturation(intensidad de color):
0% -> Escala de grises
100% -> Color puro
El tercer valor es Lightness(Luminocidad de color):
0% -> Negro
100% -> Blanco

Unidades de Medida

1- Absolutas (Su valor no cambia, son unidades del mumdo real)
- pc, cm, mm, in, Q
- pt(1/72in)
- px (1/96in)

2- Relativas (Su valor es relativo a un contexto)
- em, rem, ex, ch(al tamano de la fuente)
- em: Basada en la anchura de la "m" de la fuente del elemento.
- rem: Basada en la anchura de la "m" de la fuente del elemento raiz(html).
- ex: Basada en la altura de la "x" de la fuente del elemento.
- ch: Basada en la anchura del "0" de la fuente del elemento.

% - Al tamano del contenedor.
- vw, vh, vmin, vmax - al tamano del viewport.
- vw: Ancho del viewport, va del 1 al 100.
- vh: Alto del viewport, van de 1 a 100.
- vmin: Entre vw y vh toma el que tenga menor valor.
- vmax: Entre vw y vh toma el que tenga mayor valor.

Variables y Funciones

Custon Properties (Variables CSS)

Las variables nos sirven para guardar valores que luego mediante esta variable podamos utilizar valor en alguno de nuestras etiquetas. Las variables se almacenan dentro de una etiqueta y pueden aplicarse para los hijos de estas etiquetas.
Para utilizar estas variables, primero debemos almacenarlas, y lo hacemos de la siguiente manera: 
                                h3{
                                    --color-primario: rojo;
                                }

Es importante el uso de dos guiones antes del nombre de nuestra variable.
Para aplicar nuestra variable, lo tenemos que hacer llamando a la funcion var(), ej: 
                                h3{
                                    color: var(--color-primario);
                                }

Si por algun motivo, el valor guardado en nuestra variable no puede llegar a utilizarse, se recomienda siempre contar con valores de respaldo, que se colocan dentro del parentesis de la propiedad, colocando una coma luego del nombre de nuestra variable, de la siguiente manera: 
                                h3{
                                    color: var(--color-primario, red);
                                }

Si queremos que la variable pueda aplicarse en todos los elementos de nuestro html tenemos dos formas de hacerlo.
La primera es creando estas variables dentro del elemento html.
La segunda, y mas recomendable, es hacerlo dentro de la pseudoclase/selector :root

Funcion URL()

La funcion URL nos permite utilizar fuentes externas que se encuentren en linea, o otros archivos html/css que poseamos.
Como por ej, utilizar imagenes sin necesidad de descargarlas y ponerlas como background. 

                        background: url(https://linkdeprueba.com)

Funcion calc()

Esta funcion nos sirve para realizar calculos aritmericos. Por ej, nosotros podemos definir variables de tamano previamente, y luego a estas sumarlas entre si para aplicarlas a una propiedad. Por ej: 

                        h3{
                            --padding-size: 3ch;
                            --num-ch: 10ch;
                            background-color: yellow;
                            font-size: 2rem;
                            padding: var(--padding-size);
                            width: calc(var(--num-ch) + var(--padding-size)*2)
                        }

En ese caso, lo que hicimos fue definir variables para utilizar en padding, y luego dimos un numero(10) que en ese caso buscabamos usar como numero maximo de characteres. Al momento de dar el width, lo que queriamos era cubrir el total de los characteres y sumarle el padding tanto left como right a ese width, por lo cual el padding-size lo multiplicamos *2. Siempre debe haber espacios en blanco entre los var(operando) y el signo +(operador).
Esto nos sirve, por si se diera el caso de querer modificar la cantidad de ch tanto en num-ch como el padding-size, al modificar estos, se modificaria automaticamente el width. Ahora vamos con este ej pero ya aplicado.

Estilo de fuentes y textos

Estilos de fuentes

font-family: Es la familia tipografica
font-size: Es el tamano de la fuente
font-weight: Es el grosor de la fuente, el cual puede ser light, normal, hacerla negrita, etc.
font-style: Estilo de la fuente, el cual puede ser italica, normal, entre otras.
font-variant: Variante de la fuente. Tenemos normal, el cual deja el texto por defecto, o small-caps, el cual vuelve mayusculas todas las letras del texto.
line-height: Es el interlineado del texto

Fuentes externas

Para poder usar fuentes que no tenemos en nuestro sistema y que hayamos descargado, podemos utilizar @font-face para crear una nueva familia de fuentes con la fuente descargada. La sintexis para hacer esto es: 

                            @font-face{
                                font-family: "Nombre que querramos poner"
                                src: url("Ubicacion del archivo") format("woff") -woff significa Web Open Font Format-
                            }
                            h3{
                                font-family: "Nombre que le hayamos puesto a la family font"
                            }

Font display

Auto - Nos permite que el navegador utilice el metodo predeterminado que suele ser block.
Block - Oculta brevemente el texto hasta que la fuente haya sido descargada por completo.
Swap - Indica al navegador que utilice la fuente alternativa para mostrar el texto hasta que la fuente personalizada se haya cargado por completo.
Fallback - Es una mezcla de auto y swap.
Opctional - Ocultara el texto, luego lo cargara con la fuente alternativa y finalmente aplicara la fuente personalizada.
Para cargar de manera mas rapida las fuentes que utilicemos de manera online, ademas de agregar el href en el head, tambien debemos agregar: 

                            <'link rel="preconnect" href="url de la fuente que vamos a usar">
                            <'link rel="preconnect" href="url de la otra fuente que vamos a usar" crossorigin>

text-transform: Diferentes tipos de transformaciones que podemos aplicar al texto, como colocar todas las primeras letras en mayuscula, entre otras.

letter-spacing: espacio entre cada letra.

word-break: sirve para romper las palabras y asi que puedan entrar dentro de la etiqueta donde se encuentren.

Fondos en CSS

Backgroud-color: Define el color de fondo del elemento.
Background-image: Define la imagen de fondo del elemento.
Background-repeat: Define como se repite la imagen de fondo en el elemento.
Background-size: define el tamano de la omagen de fondo, primer valor x, segundo valor y.
-- cover: cambia el tamano de la imagen de fondo para asegurarse de que permanezca completamente visible.
-- contain: cambia el tamano de la imagen de fondo para asegurarse de que el elemento este completamente cubierto.
Background-position:
-- define la posicion de la imagen de fondo, primer valor x, segundo y, si no se especifica un segundo valor este sera center.
-- Aparte de valores numericos podemos indicar el posicionamiento con las palabras: center, top, left and right.
Background-clip: Define cuanto debe extenderse el fondo dentro del elemento.
Background-origin: Define el origen de la imagen de fondo.
Background-attachment: Define como se comportara la imagen de fondo al desplazarse por la pagina.
