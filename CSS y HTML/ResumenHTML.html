El elemento raiz de cada pagina web es la etiqueta <'html'>. Toda la informacion de nuestra pagina web estara dentro de esta etiqueta. Luego de html, tenemos 2 elementos principales: HEAD y BODY. Dentro de la etiqueta head va la parte no visible de nuestra pagina, son elementos importantes para la pagina pero no forman parte de la pagina visible. Aqui podemos colocar el link de style para darle forma con codigo css, podemos poner el titulo de la pagina el cual aparecera en el nombre de la ventana y podemos poner tambien el lenguaje que tiene nuestra pagina.



Body

Dentro del body es donde nosotros comenzamos a armar nuestra pagina, dentro de el tendremos nuestros encabezados, cuerpo de la pagina, menus, distintas vinetas donde volcaremos toda la informacion visible de la misma. Si queremos agregar parrafos a nuestro body, lo hacemos con la etiqueta <'p'>



Elemento Main

El contenido principal de nuestra pagina se va a encontrar dentro de la etiqueta <'main'>. El area principal del contenido consiste del contenido que esta directamente relacionado, o se expande sobre el tema central de un documento o la funcionalidad central de una aplicacion. No debe haber mas de un contenido main en un documento y este no debe ser descendiente de un elemento, como por ej de un article, aside, footer, header o nav.



Listas

Dentro de los elementos que podemos colocar en el body de la pagina tenemos las listas. Las listas pueden ser ordenadas(ol) o desordenadas(ul). Para colocar una lista, primero debemos colocar la vineta <'list'> y dentro de ella al tipo de lista que queremos armar <'ol'> o <'ul'>. Dentro de estas vinetas colocaremos nuestros elementos de lista dentro de vinetas <'li'> de listas.

Imagenes

Otro de los elementoso que tenemos para nuestra pagina son las imagenes. Para colocar imagenes debemos utilizar la vineta <'img'> seguido de los siglas src="" en la cual colocaremos el link de la ubicacion dentro de nuestro ordenador de la imagen. A la imagen le podemos dar altura con width y el ancho con height.





Links

Para agregar links a nuestra pagina web utilizamos la etiqueta <'a'> seguido de un href(que refiere a hyper reference) seguido de un signo ="" y el link en custion, comenzando por https. Entre medio de ambas etiquetas colocaremos algo que haga referencia al link como el nombre de la pagina, o cualquier cosa que nosotros decidamos, y al hacer click sobre este se los redirigira hacia la direccion del link. Si dentro de esta etiqueta colocamos el atributo 'target'='blank' logramos que al presionar en el link este se abra en otra pestana y nuestra pagina principal se mantenga. Otro atributo para los links es rel="noopener noreferrer" el cual nos permite evitar ataques llamados tapnabbingYoutube Podemos crear tambien enlaces internos para navegar dentro de nuestra pagina. Para esto, a cada etiqueta hay que agregarle un ID por el cual navegaremos, una vez asignado el id, creamos la etiqueta <'a'> y en la href="#body" colocamos una almohadilla y el id que hayamos asignado a la etiqueta. Tambien podemos convertir imagenes en links, para esto solamente debemos rodear la etiqueta de la imagen con una etiqueta de apertura de enlace(<'a'>) y una etiqueta de cierre()

Tablas

Las tablas tienen estructuras que se dividen en cabeza de la tabla y cuerpo de la tabla. Dentro de la cabeza iran los 'Titulos' a los cuales describiremos dentro del cuerpo de la misma. Formularios

Los formularios son elementos muy importantes a la hora de crear un sitio web, ya que mediante ellos recopilamos informacion del usuario. Este elemento permite que el ususario ingrese diferentes tipos de informacion con diferentes tipos de entrada. Para colocar un formulario utilizamos la vineta <'form'> seguido de una vineta <'input'>. Para que estos datos que completamos en el formulario sean enviados a un servidor en el cual trabajemos debemos colocar dentro de la vineta de apertura <'form'> 'action=""' y dentro de las comillas colocar la direccion del archivo del servidor que recopilara estos datos. Como por el momento no trabajamos con un servidor lo completaremos con '/enviar-respuesta' Si dentro de nuestro formulario queremos que sea requerida una respuesta si o si para poder ser enviado, utilizamos el atributo 'required' dentro de la etiqueta del input. Tenemos diferentes tipos de formularios y a continuacion tenemos algunos ejemplos de estos. Dentro de estos formularios tenemos diferentes atributos que podemos colocar, como por ej placeholder el cual coloca un texto dentro del formulario en el cual indicaremos que debe colocar el usuario. En el nombre de cada formulario colocaremos un nombre por el cual querramos llamarlo al momento de editarlo mediante css o usarlo mediante un automatizador.

ATRIBUTO CHECKED
El atributo checked nos permite colocar alguna de las casillas de nuestros botones radio figura marcada por defecto al ingresar en la pagina. Este atributo se coloca dentro del elemento input.

ATRIBUTO VALUE
Este atributo nos permite enviar al servidor el valor del elemento seleccionado dentro de nuestro formulario. Este atributo se coloca dentro del elemento input.


ATRIBUTO READ ONLY
Este atributo vuelve nuestro input de solo lectura, lo cual no nos permite modificar este texto.


DISABLE
Pone como deshabilitado el input, por lo cual figurara de un color gris oscuro, lo cual no nos deja modificar su contenido


FORM ACTION Este form nos permite enviar la informacion de los input a la direccion que coloquemos dentro de la etiqueta action.
La forma de colocarlo es 

 <'form action ="https://paginadeejemplo.com">


METODO GET/POST
Al momento de enviar datos a una URL mediante el action, el metodo que viene por defecto es GET. Lo que hace este metodo es enviar los datos, por ej, de usuario y password a esta URL, pero estos datos quedan expuestos en la misma.
Por ej, si yo envio datos a la URL www.ejemplo.com/ al enviar estos datos con este metodo, esta URL quedara www.ejemplo.com/?usuario=usuario&password=password dejando nuestros datos expuestos.
Con el metodo POST, estos datos quedan ocultos y no se mostraran en la URL a la cual los enviemos.


ATRIBUTO AUTOCOMPLETE
Si colocamos el atributo autocomplete dentro de nuestro form, tenemos la opcion de ponerlo como ON u OFF. Teniendolo encendido, al hacer click en el espacio de texto donde colocamos nuestro usuario, nos dara la oportunidad de autocompletar con los usuarios que hayamos agregado anteriormente. Si queremos evitar esto, colocaremos off en el autocomplete.

Data Atributes


Los data atributes son atributos que podemos crear y personalizar en nuestras etiquetas que luego usaremos para entre otras cosas registrar datos, agregar otros atributos a esa etiqueta, mandar a llamar, etc.
Para poder utilizar un data atribute es necesario dentro de la etiqueta a la cual se lo querramos agregar iniciar con un data seguido de un guion y finalizando con el nombre del atributo que deseemos crear. por ej: 

                    
                        <'h2' data-name="titulo"> Titulo <'/h2'>

Etiquetas Semanticas Estructurales

H1 - solo puede existir uno por pagina.
Div - es una etiqueta que no tiene semantica. Es una etiqueta de bloque. Una etiqueta de bloque es aquella que genera saltos de linea con sus etiquetas hermanas.
Header - Cabecera de un sitio web o de una seccion.
Main - Define la seccion principal del documento html.
Footer - Pie de pagina de una web.
Nav - Representa una navegacion.
Article - Representa una seccion autocontenida.
Aside - Contenido complementario o secundario(por ej, en un aside es valico colocar publicidad).
Section - Representa una seccion de contenido generico.
Address - Representa una informacion de contacto.
Las etiqueta semanticas son muy subjetivas, dependen del contenido.

Etiqueta de bloque vs etiquetas de linea

La etiqueta de bloque por excelencia es la div; la etiqueta de linea por excelencia es la span(es un contenedor generico en linea, no tiene carga semantica). Una etiqueta de linea solo ocupa el espacio necesario que tiene su contenido.

Scripts en html

La primera forma de llamar un documento scrip es mediante su etiqueta <'script>. Esta se coloca al final de nuestro archivo html, justo antes del final de body.
Otra forma, es a travez de un archivo .js, que se agrega de la misma manera, entre etiquetas <'script> con la diferencia que dentro de la etiqueta ponemos un src(source) con la ruta donde se encuentra nuestro archivo .js.
Ej: <'script src="../CSS y HTML/archivo.js"><'/script>.

Audio y Video

A nuestras web podemos agregarles archivos multimedia, los cuales deberan contar con ciertos atributos para poder ser visualizados dentro de nuestra web.
Para ello, debemos utilizar la etiqueta <'audio> o la etiqueta <'video>, las cuales al igual que las imagenes, dentro de la etiqueta de apertura van acompanadas de un src=""(source) con la direccion en la cual se encuentra nuestro archivo.
Para que nuestro archivo audio sea visualizado, la etiqueta debe ir acompanada del atriburo booleano llamado 'controls' lo cual hara que aparezca un reproductor de audio que nos permitira reproducir el archivo. El diseno de este reproductor de audio variara dependiendo del navegador.
Para nuestro audio tenemos algunos atributos booleanos que podemos aplicar, como por ej autoplay, el cual hace que cada vez que ingresemos a la pagina el audio se reproduzca, tambien loop, para que el audio se reproduzca en loop. Otro que podemos utilizar es mute, el cual mutea por defecto al audio, tambien tenemos preload, el cual ayuda para que cuando el usuario le de play al audio este ya este pre cargado.



Iframes

El iframe es un contexto de navagacion anidado, el cual nos permite incrustar otra pagina HTML en la pagina actual.
Debido a que cada contexto de navegacion es un entorno de documento completo, cada iframe en una pagina requiere mas memoria y otros recursos informaticos.
Hay varias pagias con proteccion la cual hace que no podamos utilizarlas dentro de los iframes.


Meta etiquetas para Redes Sociales

Las meta etiquetas son muy importantes para nuestras webs, estas se encuentran dentro del contenido no visible de nuestro codigo HTML, es decir, dentro de la etiqueta head.
Dentro de las meta etiquetas, contamos con Description, la cual da una breve descripcion que nosotros pongamos acerca de nuestra web la cual aparecera en los buscadores(por ej Google), y ayudara a que nuestra web sea mas facil de encontrar.
Tambien tenemos las meta etiquetas 

                <'link name="icon" href="direccion del icono">
                <'link name="apple-touch-icon" href="direccion del icono">

Estas etiquetas se utilizan para darle iconos a la pestana de nuestra web, como asi tambien un icono a nuestra web dentro de los dispositivos moviles.
Tambien contamos con meta etiquetas para el uso de nuestra web dentro de redes sociales, que por ej, crearan tarjetas con resumenes de nuestra web al colocar el link dentro de ellas.
