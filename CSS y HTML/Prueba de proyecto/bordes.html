Personalizar bordes de imagenes

Las imagenes que colocamos dentro de nuestra web pueden ser personalizadas, como por ej cambiar el ancho[widht], el alto[height], podemos agregarle bordes, entre otras cosas.
El ultimo mencionado, el borde, tiene distintos tipos de personalizaciones. Primero comenzamos agregando el border-width, luego podemos agregarle un color, stilo, entre otras cosas. Un borde puede tener diferentes estilos en cada uno de sus lados y estos se agregan solamente poniendo el nombre del estilo separado por un espacio y se van a colocar en orden: arriba, derecha, abajo, izquierda.

Imagenes redondeadas

Dentro de todas las personalizaciones que podemos agregarle a nuestras imagenes, una de ellas es la de agregarle bordes redondeados. Si aprovechamos esta funcion, podemos lograr que nuestra imagen tome forma redondeada dandole un valor de 50% a este border-radius.

Padding

Padding es el espacio entre el contenido y el borde del elemento. El borde es escensial para conocer la distancia entre un elemento y otro. Podemos aplicar diferentes padding a cada lado del elemento con el atributo sumado a top, right, left or bottom. El tamano que apliquemos se aplicara al lado que elijamos.
Por ej: padding-top: 10px. (Se aplicaran 10pixeles a la parte superior).
Para poder aplicar padding a cada lado de nuestro elemento en una sola linea, podemos hacerlo todo dentro del elemento padding agregando los valores en sentido a las agujas del reloj, tomando top el primer valor, right el segundo, bottom el tercero y lefl el ultimo.
Por ej: padding: 10px 15px 5px 10px(se le aplicaran 10 pixeles a top, 15 pixeles a right, 5 pixeles a bottom y 10 pixeles a left).

Margin

Margin es la distancia entre el borde de un elemento con el borde de otro elemento.
Al igual que con padding, a margin se le pueden aplicar valores llamando a cada lado individualmente(top, right, left, bottom) o haciendolo en una sola linea(se aplica siguiendo las agujas del reloj).
