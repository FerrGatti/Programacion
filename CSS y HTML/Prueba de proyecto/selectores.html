Selectores CSS

Los selectores son la forma en la cual nosotros seleccionamos el tipo de elemento a la cual se le va a aplicar el estilo indicado en nuestro archivo css.
Como ya indicamos ateriormente, contamos con varios selectores basicos.


Selector de tipo

Selecciona a los elementos por su tag name.
Por ej 'h1' selecciona a todos los elementos <'h1'>


Selector de clase

El selector de clase llama a los elementos por su class name y se indica de la siguiente manera: '.texto'. En este caso, llamaria a todos los elementos con el class name 'texto'.


Selector de ID

Selecciona al elemento basandose en su valor de atributo ID. Se indica de la siguiente manera: #nombre. Se aplicaria a todos los elementos con el valor de ID 'nombre'.


Selector universal

Selecciona a todos los elementos. Opcionalmente puede estar restringido a un espacio de nombre especifico o a todos los espacios de nombres.
Se aplica de la siguiente manera: *. Se aplicara a todos los elementos del documento.


Selector de atributo

Selecciona a los elementos basandose en el valor de un determinado atributo.
Ej: img[alt] Se aplicara a todas las imagenes que contengan el atributo 'alt'
Ej: [type="radio"] De esta manera seleccionamos a todos los elementos de tipo radio.
