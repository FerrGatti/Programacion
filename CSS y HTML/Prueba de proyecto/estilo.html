Estilo en linea de codigo

Una de las formas de poder aplicar estilos a nuestra pagina es hacerlo directamente desde la linea de codigo. Este estilo debemos aplicarlo en la etiqueta de apertura a la cual lo aplicaremos en forma de atributo colocando 'style'. Dentro de las diferentes modificaciones que podemos hacer estan el color, la fuente, el tamano, la ubicacion en la pagina, etc.


Estilo en head

Otra de las maneras que tenemos de aplicar estilos a nuestra pagina es a travez de la etiqueta <'style'> colocada dentro de head. Dentro de esta etiqueta colocaremos la etiqueta, nombre, id, tipo o class que buscamos darle estilo y entre {} iremos modificando el color, la fuente, el tamano de la fuente, etc.
Por ej:

h1{
color: red;
font: arial;
font-size: 5px;
}

De esta manera, le aplicamos estilo al texto dentro de la etiqueta h1.
A diferencia del aplicado en linea, cuando utilizamos el style en head, este estilo se aplica a todos los textos que compartan etiqueta, por ej, si tenemos mas de un header 1, el estilo se aplicara a todos y cada uno de ellos.


Creando un archivo css

Otra de las maneras de aplicar estilo a una pagina web es creando un archivo .css en el cual, de la misma manera que aplicando stilos dentro de head, llamaremos a cada etiqueta, nombre, id, type, etc, de manera individual y mediante {} aplicaremos cambios de color, fuente, etc.
Para utilizar este archivo css, en la parte de head vamos a utilizar una nueva etiqueta llamada <'link'> seguida de un href en el cual colocaremos la direccion en nuestra pc donde se encuentre el archivo css a utilizar. Por ej: href="../CSS y HTML/style.css" rel="stylesheet"
Para evitar que todos los estilos se apliquen a la etiqueta a la cual llamamos, como por ejemplo h1, debemos aplicar estos estilos utilizando el nombre o el class de la etiqueta, el cual logra que estos cambios se apliquen especificamente a estos.
