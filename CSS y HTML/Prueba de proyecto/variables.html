Variables en CSS

Para crear una variable, primero debemos colocar dos guiones(--) seguido del nombre de la variable y del valor de esta.
Una de las funcionalidades que podemos encontrarles es dandole un valor de color, por ej, --color: red. Luego cuando queramos asignar este color en alguna etiqueta, solamente colocamos el atributo, seguido de var y entre parentesis el nombre de la variable.
Ej: h2{
background-color: var(--color)
}
En este caso, el color de fondo de h2 seria el valor de la variable previamente asignado. Esto nos es muy util ya que, al asignarle el valor de nuestra variable a diferentes elementos, si buscamos cambiar el valor de la variable solamente modificaremos su valor y este se aplicara a todos los elementos.

Valores de respaldo

Para prevenir errores en la asignacion de variables, en caso de que la variable no se consiga o no este correcta, el valor de respaldo nos ayudaria a cubrir la ausencia del valor principal. Para agregarlo, solamente colocamos una coma, seguido de un espacio y el valor de respaldo.
Ej: h1{
background-color: var(--color, negro)
}
