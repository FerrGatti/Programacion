Dark mode en imagenes

Podemos utilizar el filtro invert para poder aplicar modos claros u oscuros.
Para esto, debemos tener en cuenta que al aplicar el filtro invert en la etiqueta html o body, este se aplicara a todos nuestros elementos como por ej las imagenes, luego debemos volver a aplicarlo directamente a estas para que vuelvan a su filtro original.
Por ej, si aplico una clase Dark-mode en html, esto va a invertir los colores de todo, incluyendo las imagenes.
Una vez aplicado, debemos llamar a esta clase seguido de img para que aplique a todas las imagenes, y a este aplicamos nuevamente el filtro invert para que estas se inviertan de nuevo volviendo a su color original.
